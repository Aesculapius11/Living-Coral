这篇文章会简要记录本站的技术栈选择与理由：

1. 选择 **Eleventy**：纯静态输出，模板灵活，易于长期维护。
2. 选择 **Tailwind CSS**：原子化样式让开发迭代极快，且便于做暗黑模式与响应式。
3. 动画库选用 **AOS**：即用型滚动动画，轻量且无侵入。

最后，通过 GitHub Pages、Vercel 或 Netlify 就能一键部署。
